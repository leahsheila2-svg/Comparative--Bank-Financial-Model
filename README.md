# Comparative Bank Financial Model

## Summary: 

This repository presents a comprehensive comparative financial model developed to evaluate and benchmark the financial performance, operating efficiency, profitability, balance sheet strength, liquidity position, and valuation of two listed commercial banks using publicly available financial statements.

The model is designed to replicate the analytical workflow undertaken by corporate finance professionals, investment analysts, equity researchers, and institutional investors when comparing competing financial institutions within the same industry.

Rather than analysing a single institution in isolation, the model applies a structured peer comparison framework that enables investors to identify relative strengths, operational efficiencies, valuation differences, and potential investment opportunities across comparable banks.

Historical financial information is standardized into a consistent modelling framework before being transformed into meaningful financial ratios, trend analyses, and valuation metrics. The result is a decision-support tool that combines accounting analysis with financial modelling to facilitate informed investment decisions.

The workbook demonstrates how comparative financial analysis extends beyond reviewing financial statements by translating accounting data into actionable insights regarding profitability, capital allocation, earnings quality, operating efficiency, shareholder returns, and long-term value creation.
